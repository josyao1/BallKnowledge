# Ball Knowledge

NBA + NFL sports trivia game with multiple modes — guess rosters, unscramble names, identify players from stats, and more.

## Official Link

[ball-knowledge-delta.vercel.app](https://ball-knowledge-delta.vercel.app)

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

App runs at `http://localhost:5173`. No backend required — all data is served as static JSON from Vercel CDN.

## Game Modes

- **Roster Royale** — Solo or multiplayer. Guess players from a random or manually selected team/season. Host can assign adaptive score multipliers (1×–4×) per player in the lobby.
- **Division Mode** — Multiplayer. Each player is assigned a different team from the same division.
- **Career Mode** — Identify a player from their season-by-season stat line.
- **Name Scramble** — Solo or multiplayer. Unscramble a shuffled player name. Multiplayer: race-to-answer with positional scoring (1st=5pts, 2nd=3pts, 3rd=2pts, 4th=1pt) and a 30-second pressure timer; first to reach the point target wins.
- **Roll Call** — Open-ended multiplayer: name anyone who ever played for a franchise.
- **Lineup Is Right** — Solo or multiplayer. Pick players whose stats add up as close as possible to a target cap without going over. 5 rounds, same team per round. Choose from multiple stat categories (PTS, AST, REB, PRA, GP, etc.). Results show an "optimal last pick" hint. Multiplayer is simultaneous — all players pick at the same time each round.
- **Starting Lineup** — Solo or multiplayer. See an NFL formation with player blobs encoded as college logos, jersey numbers, or draft pick numbers — guess the team. Solo includes a bonus round to name individual players (+1pt each, 60s timer). Multiplayer: first correct answer earns 3pts, others within the 30-second window earn 1pt; first to 10/20/30pts wins. NFL only for now (NBA planned).

## Tech Stack

- React 19 + TypeScript + Vite
- Tailwind CSS + Framer Motion
- Zustand (state management)
- Fuse.js (fuzzy search autocomplete)
- Supabase (multiplayer lobbies + leaderboard)
- Static JSON on Vercel CDN (all roster + career data)

## Environment Variables

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

Supabase is optional — the app works fully offline without it (multiplayer and leaderboard features are disabled).

## Static Data

All game data lives in `public/data/` and is pre-generated by scripts in `scripts/`:

| Path | Contents |
|---|---|
| `public/data/rosters/` | ~775 NBA roster files (2000–01 through 2025–26) |
| `public/data/players/` | NBA league-wide autocomplete per season |
| `public/data/nfl/rosters/` | ~754 NFL roster files (2000–2025) |
| `public/data/nfl/players/` | NFL league-wide autocomplete per year |
| `public/data/nba_careers.json` | 736 NBA players, full season-by-season stats |
| `public/data/nba_lineup_pool.json` | 410 recent NBA players for Lineup Is Right + Scramble |
| `public/data/nfl_careers.json` | 584 NFL players (QB/RB/WR/TE) |
| `public/data/nfl_lineup_pool.json` | Recent NFL players for Lineup Is Right + Scramble |
| `public/data/nfl/starters_2025.json` | NFL 2025 starting lineups (32 teams, offense + defense) |
| `public/data/nba/starters_2026.json` | NBA 2025-26 starting fives (30 teams) with PPG hints |

To regenerate or add new seasons:

```bash
cd scripts

# NBA rosters (resume-safe, skips existing files)
python generate_nba_rosters.py

# NFL rosters (fast — uses local nfl_data_py)
python generate_nfl_rosters.py

# Career stats (full rebuild)
python generate_nba_careers.py
python generate_nfl_careers.py

# NBA career incremental update (append new seasons only, much faster)
python update_nba_careers.py --years 2025 2026

# Lineup Is Right expanded player pool (recent players, resumable)
python -u generate_nba_lineup_pool.py --resume
cp data/nba_lineup_pool.json ../public/data/nba_lineup_pool.json
```
